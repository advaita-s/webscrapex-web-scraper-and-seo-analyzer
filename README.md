# webscrapex-web-scraper-and-seo-analyzer
WebScrapeX is a modern AI-powered web scraping and SEO analysis tool. Extract articles, product details, prices, structured data, and generate smart summaries. Includes SEO insights, job history, and a clean dashboard UI. Built with Python (FastAPI), SQLModel, SQLite, Next.js, React, Axios, TailwindCSS.
